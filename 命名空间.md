### **定义命名空间**

```c++
namespace namespace_name {

}
```



**命名空间只要是名字是一样的就算一个**

可以在a.cpp里在namespace A里加东西，在b.cpp里也可以在A里加东西



### **using**

**using只在你写的作用域内生效 a.cpp里using的在b.cpp里就不起效了**

#### 1.只引入名空间里的一个符号

```c++
namespace A {
	void f() { cout << 1 << endl; }
}
int main() {
	using A::f;
	f();
}
```



#### 2.引入整个名空间

```c++
namespace A {
	void f() { cout << 1 << endl; }
}
int main() {
	using namespace A;
	f();
}
```

**c++强制类型转换运算符用法**

```c++
强制类型运算符<要转换到的类型>(要转换的表达式)
```



### static_cast

**只能用于整形，浮点数和字符之间的转化，或者对象重载了强制类型转换运算符**

不能用于不同类型指针之间转化以及整形和指针之间转化

```c++
int a;
char b=static_cast<char>(a);
```



### **const_cast**

**四个类型转化里唯一能去除const属性的运算符**

```c++
const string s = "111";
string* ss = const_cast<string*>(& s);
```



### reinterpret_cast

**支持指针之间的转化和整形和指针之间的转化**

```c++
int a = 1;
double* b = reinterpret_cast<double*>(&a);
cout << *b;
//
int a = 1;
double* b = reinterpret_cast<double*>(a);
cout << b;
```



### dynamic_cast

**用于多态的派生类对象**

```c++
class A {
	virtual void func()=0;
};
class B :public A {
	void func() {
		cout << 1;
	}
};
int main() {
	A* a = new B;
	B*b = dynamic_cast<B*>(a);

}
```



### 类型转换运算符函数

```c++
class A {
public:
	int a;
	operator double() {
		return static_cast<double>(a);
	 }
};
//可以加上explicit 防止强制类型转换
explicit operator double() {
		return static_cast<double>(a);
}
```

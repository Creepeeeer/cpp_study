### 全局函数作为友元

正常在类外不能访问类的private成员，如果在类里声明了这个全局函数是友元的话，这个函数里的类对象就能访问类的私有成员变量

```c++
class A {
	friend void f(const A& x);
private:
	int a;
};
void f(const A& x) {
	cout << x.a << endl;
}
```



### 类做友元

正常一个类的private成员不能被另外一个类访问，如果在这个类里已经声明了另外一个类是这个类的友元，那么就可以访问私有成员

```c++
class A {
	friend class B;
private:
	int a;
};
class B {
public:
	void f(const A& x) {
		cout << x.a << endl;
	}
};
```



### 成员函数做友元

如果让一个类做友元那么一个类都能访问所有成员，成员函数做友元就相当于只有这个函数能访问

```c++
class A;
class B {
public:
	void f(const A& x);
};
class A {
	friend void B::f(const A& x);
private:
	int a;
};
void B::f(const A& x) {
	cout << x.a << endl;
}
```

**顺序不能错 先声明A 然后定义B，然后定义A，最后再定义B::f**
